# FLEXIBLE PACKAGING ERP - COMPLETE IMPLEMENTATION GUIDE
**Sistrade-Inspired System | Customer Contact → Product Delivery**

## Tech Stack
- **Frontend:** React 18+ TypeScript, Tailwind CSS, shadcn/ui
- **Backend:** Node.js, Express, TypeScript  
- **Database:** PostgreSQL 15+
- **Auth:** JWT + Refresh Tokens
- **Real-time:** Socket.io
- **State:** Redux Toolkit

## DEPLOYMENT STRATEGY: MODULE-BY-MODULE

### How to Deploy Incrementally to GoDaddy

**Step 1: Setup (One Time)**
```bash
# On your PC
git init
git remote add origin <your-repo-url>

# On GoDaddy VPS (SSH)
cd /var/www
git clone <your-repo-url> flexpack-erp
cd flexpack-erp
npm install
npm run build
```

**Step 2: Deploy Each Module**
```bash
# On PC - after completing a module
git add .
git commit -m "feat: add Module X - [functionality]"
git push origin main

# On GoDaddy VPS
cd /var/www/flexpack-erp
git pull origin main
npm install  # if new dependencies
npm run migrate  # run database migrations
pm2 restart flexpack-erp  # restart app
```

**Database Migration Pattern:**
```bash
# Create migration for each module
npm run migration:create -- add_module_X_tables

# In migration file
exports.up = (pgm) => {
  pgm.createTable('new_table', { ... });
  pgm.addColumn('existing_table', 'new_column', { ... });
};

exports.down = (pgm) => {
  pgm.dropTable('new_table');
  pgm.dropColumn('existing_table', 'new_column');
};

# Run migration
npm run migrate up
```
